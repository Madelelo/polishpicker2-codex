# PolishPicker

PolishPicker is a website for cataloging your full nail polish collection and exploring nail art ideas through an interactive front-end interface.

## Features

- Track your personal nail polish inventory
- Store brand, color, finish, and notes for each polish
- Browse nail art inspirations by style, difficulty, and vibe
- Mix and match colors to plan looks

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.

## Project Structure

- `index.html` - main app UI
- `styles.css` - visual styling
- `app.js` - client-side interactivity

## Vision

PolishPicker is designed to be both a personal polish tracker and a creative discovery tool for nail art exploration.
